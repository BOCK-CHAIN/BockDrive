# Bock Drive

A Google Drive clone built with React, Firebase, and TailwindCSS. Store, manage, and share your files securely in the cloud.

## Features

- 📁 File and folder management
- 🔒 Secure authentication
- ⭐ Star important files
- 🗑️ Trash bin functionality
- 📱 Responsive design
- 🔍 File search and filtering

## Tech Stack

- React
- Firebase (Authentication, Firestore, Storage)
- TailwindCSS
- React Router
- Zustand
